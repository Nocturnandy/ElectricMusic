# Electric Music — Record Label Sites

Twin record labels under the **Electric** banner, based in Naarm / Melbourne.

- **Sundara** — sunrise twin. Organic, melodic, balearic house. *Stay For Sunrise.*
- **Midnite Arcade** — after-dark twin. House, techno, electro. *Play Past Midnite.*
