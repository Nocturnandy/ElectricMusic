# Midnite Arcade — Vector Element Pack

Clean, scalable versions of the core Midnite Arcade brand parts, traced from the
4K master document (`Midnite_Arcade_Master_Doc_4x.png`) and the label cards.

## What's inside

`/SVG` — infinitely scalable vectors (use for print, merch, large format, web)
`/PNG` — 2048px transparent rasters (use for quick drops into decks, social, mockups)

| Element | Notes |
|---|---|
| MA_Wordmark | Primary "MIDNITE ARCADE" hand-drawn lockup, cream |
| MA_Ghost | Mascot ghost |
| MA_Bolt | Lightning bolt, gold |
| MA_Smiley | Classic smiley, gold |
| MA_Star | 4-point twinkle, cream |
| MA_Globe | Globe icon, cream |
| MA_Checker | Checker/hash motif, cream |
| MA_Palette | Brand colour swatches with hex values |

## Colours

Black `#0D0D0D` · Cream `#E7E2D6` · Red `#E23125` · Green `#34923D` · Blue `#1C5486` · Gold `#E9A81C`

## Notes

The hand-drawn roughness is preserved deliberately — these are faithful traces, not
redrawn type, so they still read as the original marks at any size. Recolour any SVG
by editing the `fill` value in the file.
